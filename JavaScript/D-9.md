## Intervals

```javascript
function sayHello(){
    console.log("Hello")
}

setInterval(sayHello, 2000); // 2000ms 간격으로 실행
```


## Timeouts and Dates


```javascript
const clock = document.querySelector("h2#clock");

function getDate(){
    const date = new Date();
    clock.innerHTML = (`${date.getHours()} : ${date.getMinutes()}:${date.getSeconds()}`);
}
```
getDate();
